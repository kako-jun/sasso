# Pixel Calculator (React + Vite)

モダンなReact構成で白黒ピクセル調の電卓UIを実装しています。基本的な四則演算、パーセント、符号切り替えに対応しています。

## セットアップ

1. 依存関係のインストール:

   ```bash
   npm install
   ```

2. 開発サーバーの起動:

   ```bash
   npm run dev
   ```

3. 本番ビルド:

   ```bash
   npm run build
   npm run preview
   ```

## コードスタイル

- TypeScript + React (Vite)
- ESLint (flat config) を使用しています。`npm run lint` でチェックできます。

## 主要ファイル

- `src/App.tsx`: 電卓ロジックとレイアウト
- `src/App.css`: ピクセル調デザインのスタイル
- `src/index.css`: 全体のベーススタイル
