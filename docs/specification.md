# Sasso - Classic Macintosh Style Calculator

## 概要

SassoはClassic Macintosh System 7風のデザインを再現した電卓アプリケーションです。

## 技術スタック

- React 18
- TypeScript
- Vite
- CSS (vanilla)

## UI仕様

### デスクトップ背景
- 45度の斜線パターン（白地に黒線）
- パターンサイズ: 4px x 4px

### メニューバー
- 白背景、下部に黒線
- 項目: Apple logo, File, Edit, View, Special
- 高さ: 20px

### ウィンドウ
- 角丸: 8px
- 枠線: 3px solid black
- 影: 5px 5px 0 black

### タイトルバー
- 背景: 黒
- 高さ: 24px
- 下部ボーダー: 3px solid black
- タイトル「Sasso」: 白文字、18px、中央揃え
- クローズボックス: 16px x 16px、白背景、左端配置

### ウィンドウコンテンツ
- 背景: 互い違いのドットパターン（1px黒ドット、4px間隔）
- パディング: 12px 14px 12px 12px

### ディスプレイ
- 白背景
- 枠線: 2px solid black
- 内側シャドウ: inset 2px 2px 0 #888
- テキスト: 右揃え、20px

### キーパッド
- CSS Gridレイアウト: 4列
- 列間隔: 4px
- 行間隔: 8px

### ボタン
- サイズ: 34px x 28px
- 背景: 白
- 枠線: 2px solid black
- 影: 2px 2px 0 black
- フォントサイズ: 16px
- クリック時: 影なし、2px右下に移動

### 特殊ボタン
- 0ボタン: 2列幅、テキスト左揃え（padding-left: 12px）
- +ボタン: 2行高、テキスト下揃え

## ボタン配置

```
| C | E | = | * |
| 7 | 8 | 9 | / |
| 4 | 5 | 6 | - |
| 1 | 2 | 3 |   |
| 0     | . | + |
```

## 機能仕様

### 基本演算
- 加算 (+)
- 減算 (-)
- 乗算 (*)
- 除算 (/)

### 特殊機能
- C: 全クリア（表示、累積値、演算子をリセット）
- E: エントリクリア（表示のみリセット）
- =: 計算実行
- .: 小数点入力

### エラー処理
- 0除算: 「E」を表示
- オーバーフロー: 指数表記で表示

### 表示フォーマット
- 最大10桁
- 10桁を超える場合は指数表記（4桁精度）

## キーボード操作

| キー | 機能 |
|------|------|
| 0-9 | 数字入力 |
| + | 加算 |
| - | 減算 |
| * | 乗算 |
| / | 除算 |
| Enter, = | 計算実行 |
| Escape, C, c | 全クリア |
| Backspace, Delete, E, e | エントリクリア |
| . | 小数点 |
